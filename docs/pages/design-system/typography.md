# 排版 (Typography)

定義了網站使用的字型家族、字級與行高。

## 字型家族 (Font Family)

我們主要使用以下字型堆疊：

- **Base**: `Roboto`, `NotoSansTC`, `微軟正黑體`, `Arial`, `sans-serif`
- **Mono**: `FiraCode`, `Roboto`, `sans-serif`
