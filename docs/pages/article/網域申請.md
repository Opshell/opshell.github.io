# 網域廠商評估與申請筆記

## 1. 現況分析
* **原網域**：`opshell.com` 因信用卡過期未續約，已被他人搶註。
* **需求**：建立個人技術部落格 (`opshell.me`) 與攝影集圖床 (`images.opshell.me`)。
* **架構**：GitHub Pages (前端) + Cloudflare R2 (圖床)。

## 2. 廠商評估：Cloudflare vs Squarespace
由於 Google Domains 業務已被 Squarespace 收購，目前主要考量如下：

| 比較項目 | **Cloudflare Registrar** (勝出 🏆) | **Squarespace** |
| :--- | :--- | :--- |
| **定價策略** | **成本價 (At-cost)**，不賺差價。 | 商業營利，通常較貴。 |
| **續約費用** | 低廉 (例如 `.dev` 約 $10-12/年)。 | 較高 (約 $20+/年)。 |
| **整合性** | **極佳**。買完直接整合 DNS、CDN、R2。 | **差**。需手動修改 NS 紀錄指向 Cloudflare。 |
| **適合對象** | 開發者、工程師。 | 一般商家、不寫程式者。 |

**結論**：因後續需使用 Cloudflare R2 與 CDN 服務，直接在 Cloudflare 購買網域可省去 DNS 搬遷手續且長期持有成本最低。

## 3. 網域後綴 (TLD) 選擇策略
* **`.dev`**：適合技術人員，Google 強制 HTTPS，專業感強。
* **`.me`**：適合個人品牌，兼顧技術與攝影 (About Me)，親切感高。
* **`.design` / `.studio`**：適合強調設計系統 (Design System) 或職人精神。

**最終決定**：購買 `opshell.me`。

## 4. 遇到的問題與解法 (Troubleshooting)
* **問題**：`opshell.com` 被搶註。
    * **解法**：放棄贖回（價格過高），轉而註冊更符合個人品牌定位的新後綴 `.me` 或 `.dev`。
* **問題**：信用卡過期導致服務中斷。
    * **解法**：在 Cloudflare 綁定常用信用卡，並確保開啟自動續約。
