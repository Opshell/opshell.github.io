**2. 核心穩定性與資安防護 (Core Stability & Security)**
*面對國家級機房的高規資安要求，以及後端 API 資料不可信的潛在風險：*
* **[型別安全]** 建構基於 **Zod** 的 **Schema-First** 開發流程，實作 **100% Runtime Type Safety (執行期型別安全)**，將 API 資料驗證與型別定義合而為一，徹底杜絕 `undefined` 造成的線上崩潰。
* **[防腐層設計]** 實作 **Axios Interceptors** 與 **Adapter Pattern** 作為 API 請求層的防腐層 (ACL)，統一處理全域錯誤與 **OAuth 2.0** 身份驗證邏輯，確保業務代碼純淨，不被底層通訊細節污染。
* **[狀態管理]** 針對複雜的資料依賴與網路延遲問題，導入 **TanStack Query** 接管伺服器狀態 (Server State)，以快取策略優化使用者體驗，並減少 **43%** 的細節狀態開發成本。

**3. 效能工程與監控 (Performance Engineering)**
*面對使用者年齡層廣泛且硬體設備差異大的挑戰，必須確保極致的載入速度：*
* **[載入優化]** 設計精細的 **Code Splitting (代碼分割)** 與 **Vendor Chunking (分包策略)**，配合 Gzip 壓縮，成功將首屏 Payload 體積縮減 **43%**，顯著優化 FCP (First Contentful Paint)。
* **[持續監控]** 建立基於 **Bundle Visualizer** 的體積監控機制，將 **Tree Shaking** 與依賴優化納入定期維護流程，確保專案長期迭代下仍保持輕量化。

**4. 開發者體驗優化 (Developer Experience, DX)**
